Webcache

A software compatible with most common operating systems that allows web surfers to cache and annotate web pages.

Download Desktop App

Features

WebCache allows you to conveniently save entire webpages, add annotation to the webpages, and search through all saved pages by content, annotations, or other attributes.

Capture

Ability to retrieve a web page, extract its HTML and any other related files, and save it to your computer.

Annotation

Use the highlighter tool directly on the saved web page to bring attention to relevant or important sections and the comment capability to add your notes to highlighted content.

Search

Search quickly and efficiently through saved web pages and annotations.

Web Pages

The Web is not a static entity, pages are subject to change at any given time. This can be a problem for those who want to always be able to access a page as it appears. WebCache is here to solve this problem! The application provides the ability to retrieve a web page, extract its HTML and any other related files, and save it all to your computer.

Annotations

Add
Annotation

While reading documents, you may come across an entry that you don't want to have to look for again. Use the highlighter tool directly on the saved webpage to bring attention to relevant or important sections. You can also add comments to any given highlight for seamless note taking.

Search

Search
for Content

Save a useful webpage for later reference and forget where you put it? WebCache has you covered. Through the application, users are able to search quickly and efficiently through saved web pages and annotations. Recall what was in the webpage but not what your comment said or vice versa? Rest assured that WebCache supports searching by either one!

Scrapbook Compatibility

Import Legacy Data
From Scrapbook

Scrapbook was a webcaching application for Firefox that supported many of the features that WebCache is now home to. Unfortunately for users of the app, its support has now been discontinued. If you were an old user of Scrapbook, your worries are over! WebCache allows users to import their old cached webpages, complete with added annotations. Comments and highlights associated with the page are all supported and displayed within WebCache.
